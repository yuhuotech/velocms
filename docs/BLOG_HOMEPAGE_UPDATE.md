## 启动方式

### 开发环境

```bash
npm run dev
```

启动脚本会自动：
1. ✅ 检测 3002 端口是否被占用
2. ✅ 如果被占用，强制终止占用进程
3. ✅ 在 3002 端口启动开发服务器

### 生产环境

```bash
npm run build
npm run start
```

## 技术栈

- **框架**: Next.js 15 (App Router)

- **主题**: next-themes
- **日期**: date-fns
